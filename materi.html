Cascading Style Shets
adalah bahasa yang digunakan untuk mengatur tampilan dan desain halaman website. CSS bekerja bersama HTML untuk membuat tampilan website menjadi lebih menarik, rapi, dan mudah digunakan. CSS memiliki beberapa fungsi utama, yaitu: Mengatur warna teks dan background, Mengatur ukuran dan jenis font, Mengatur layout (tata letak) halaman, Membuat tampilan menjadi responsive (menyesuaikan layar HP/laptop) dan Menambahkan animasi dan efek visual.
PENTINGNYA BELAJAR CSS
  • Belajar CSS (Cascading Style Sheets) sangat penting dalam dunia pembuatan website karena CSS berfungsi untuk mengatur tampilan dan desain halaman web agar lebih menarik dan nyaman dilihat.
